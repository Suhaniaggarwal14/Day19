Wie Sie eine Lizenz für WinRAR und RAR erwerben

Wenn Sie WinRAR und RAR nach dem Testzeitraum vom 40 Tagen weiter benutzen möchten, müssen Sie eine Lizenz von den hier aufgelisteten Vertriebspartnern erwerben. Unter www.win-rar.com können Sie die aktuelle Preisliste einsehen und eine Lizenz online erwerben.

Sobald die Registriergebühr bei uns eingegangen ist, erhalten Sie eine Registrierungsschlüsseldatei, die mit dem von Ihnen angegebenen Benutzernamen korrespondiert. Bitte geben Sie beim Kauf einer Lizenz eine gültige E-Mail-Adresse an. Die Registrierungsschlüsseldatei wird an diese Adresse versendet.

Nach der Registrierung erhalten Sie ein E-Mail, das alle weiter notwendigen Informationen enthält. Folgen Sie diesen Anweisungen. Im Folgenden stellen wir kurz den typischen Ablauf einer Registrierung dar. Die Anweisungen im E-Mail sind aktueller als in dieser Datei und sollten Vorrang haben.

Wenn Sie WinRAR verwenden, kopieren Sie die Registrierungsschlüsseldatei (rarreg.key) in das WinRAR-Programmverzeichnis oder in das Verzeichnis %APPDATA%\WinRAR. Das Standard-WinRAR-Programmverzeichnis ist "C:\Programme\WinRAR". Es kann aber bei der Installation von WinRAR durch den Anwender geändert werden. Sie können die Datei rarreg.key auch per Drag&drop auf das WinRAR-Fenster fallen lassen, um die Registrierung vorzunehmen.

Wenn der Schlüssel in einem RAR-Archiv gepackt ist, entpacken Sie bitte die Datei rarreg.key aus dem Archiv, bevor Sie die Datei in das Installationsverzeichnis kopieren. Hat das Archiv den Namen rarkey.rar, können Sie den Registrierungsschlüssel installieren, indem Sie das Archiv mit WinRAR öffnen und die Rückfrage zur Bestätigung mit "Ja" beantworten.

Sie können die Datei rarreg.key in dasselbe Verzeichnis wie das WinRAR-Installationsprogramm kopieren. Dann verwendet das Installationsprogramm die Registrierungsschlüsseldatei rarreg.key automatisch für die Registrierung, wenn keine zuvor installierten Schlüsseldateien gefunden werden.

Im Falle von RAR für Unix und RAR für macOS sollten Sie rarreg.key in Ihr Home-Verzeichnis oder in eines der folgenden Verzeichnis kopieren: /etc, /usr/lib, /usr/local/lib, /usr/local/etc. Wenn Sie möchten, können Sie die Registrierungsschlüsseldatei nach ".rarreg.key" oder ".rarregkey" umbenennen, der Name "rarreg.key" ist allerdings auch gültig.

WinRAR, RAR für die Befehlszeile für Unix und macOS verwenden jetzt das gleiche Registrierschlüsselformat, sodass Sie denselben Schlüssel für die derzeitig verfügbaren Versionen von WinRAR bzw. RAR auf allen erwähnten Plattformen verwenden können. Für WinRAR- und RAR-Versionen, die nicht die gleiche Versionsnummer wie die in diesem Distributionsarchiv enthaltenen Version haben, kann dies nicht garantiert werden.
